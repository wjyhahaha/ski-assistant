# 滑雪出行攻略生成模板

本文件为 ski-assistant 技能的攻略输出模板，覆盖国内及国际滑雪出行场景。

---

## 用户需求画像收集清单

生成攻略前，确保已收集以下信息（缺失项主动提问）：

| 维度 | 必选 | 问题示例 |
|------|------|---------|
| 滑雪水平 | 是 | 你的滑雪水平大概是？（初学/中级/高级/发烧友） |
| 出发城市 | 是 | 你从哪个城市出发？ |
| 出行日期 | 是 | 计划什么时候去？待几天？ |
| 同行人数 | 是 | 几个人一起？朋友/家庭/情侣？ |
| 预算区间 | 否 | 大概预算多少？（可不限） |
| 偏好标签 | 否 | 有什么特别偏好吗？（粉雪/公园/陡坡/风景/夜滑/亲子/美食） |
| 装备情况 | 否 | 自带装备还是需要租赁？ |

## 欧洲专属补充模块

```markdown
## 欧洲滑雪特别事项

### 互联雪区与联票策略
- **目标区域**：{如三峡谷/Ski Arlberg/Dolomiti Superski}
- **联票类型与价格**：{多日票/单区域票/全区域票对比}
- **推荐策略**：{根据用户天数和水平建议最优联票}
- **注意**：欧洲大型互联雪区通常按区域售票，提前了解区域划分可避免重复购票

### 欧洲交通方案
- **飞行**：{推荐机场，如日内瓦/苏黎世/慕尼黑/因斯布鲁克}
- **火车**：{瑞士/奥地利铁路网发达，火车直达雪场}
- **接驳巴士**：{机场到雪场的 shuttle 预订建议}
- **自驾**：{欧洲冬季自驾须知，如奥地利需购买 Vignette 高速贴，冬胎法规，雪链}
- **欧洲租车提示**：多为手动挡，自动挡需提前预订且价格更高；注意各国高速收费规则不同

### 欧洲美食体验
- **法国**：fondue（奶酪火锅）、raclette（拉克莱特）、tartiflette（奶酪土豆焗）、vin chaud（热红酒）
- **瑞士**：rösti（瑞士薯饼）、fondue、巧克力热饮
- **奥地利**：Wiener Schnitzel（维也纳炸猪排）、Kaiserschmarrn（皇帝煎饼）、Glühwein（热红酒）
- **意大利（多洛米蒂）**：pizza、pasta、polenta、南蒂罗尔特色 Knödel（面疙瘩）
- **餐饮预算**：雪场午餐约 EUR 15-25，镇上晚餐约 EUR 25-50
- **小贴士**：欧洲雪场山顶餐厅风景绝佳，建议至少体验一次阳光露台午餐

### Après-Ski 文化
- **定义**：滑雪后社交活动，是欧洲滑雪文化的重要组成部分
- **代表地**：St. Anton（MooserWirt/Krazy Kanguruh）、Ischgl、Val d'Isère
- **形式**：雪场脚下酒吧，通常下午 3-4 点开始，热红酒/啤酒 + 音乐
- **费用参考**：啤酒约 EUR 5-8，热红酒约 EUR 4-6
```

### Ikon/Epic Pass 策略
- **适用场景**：{根据用户行程天数判断}
- **Ikon vs Epic 对比**：

| 维度 | Ikon Pass | Epic Pass |
|------|----------|----------|
| 价格（早鸟） | USD 1200-1400 | USD 800-1000 |

- **特色推荐**：Colorado green chili（科罗拉多绿辣椒）、Utah 精酿啤酒、BC 省三文鱼
- **小费文化**：餐厅用餐通常需付 18-22% 小费
- **餐饮预算**：午餐约 USD 15-25，晚餐约 USD 30-60

### 北美时差与高海拔
- **时差**：美西（太平洋时间）比北京晚 16 小时，美山区晚 15 小时
- **高海拔**：科罗拉多雪场海拔通常 2500-3500m，初到注意高反
- **应对建议**：到达后第一天轻度滑行，多饮水，避免饮酒，如有症状可服用布洛芬

## 南半球专属补充模块

```markdown
## 南半球滑雪特别事项

### 反季节规划要点
- **雪季时间**：6 月 - 10 月（最佳 7-9 月）
- **适用人群**：北半球夏季想滑雪的爱好者、备赛运动员、Gap Year 旅行者
- **气候提醒**：南半球冬季气温波动大，新西兰山区天气多变，建议关注每日雪况报告

### 新西兰滑雪须知
- **签证**：中国公民需旅游签证，可网上申请，周期约 15-25 个工作日
- **驾照**：需携带中国驾照 + 英文翻译件（NZTA 认可），右舵左行
- **交通**：皇后镇/瓦纳卡为主要基地，雪场均需自驾或乘坐 shuttle
- **住宿**：皇后镇住宿选择丰富但旺季需提前预订，瓦纳卡更安静经济
- **特色体验**：蹦极、跳伞、米尔福德峡湾、萤火虫洞

### 澳大利亚滑雪须知
- **签证**：中国公民需旅游签证（600 类）
- **交通**：悉尼/墨尔本/堪培拉出发自驾前往，冬季山路需雪链
- **国家公园入园费**：进入 Kosciuszko National Park 需缴纳车辆入园费（约 AUD 29/天）
- **Skitube**：Perisher 可乘坐 Skitube 地下火车从 Bullocks Flat 进入，避免山路驾驶
```

## 攻略质量检查清单

- [ ] 所有价格已标注币种和时效性

- [ ] 住宿推荐有多个档位选择
- [ ] 包含安全提醒和保险建议（国内外均需）
- [ ] 包含美食/餐饮推荐和预算
- [ ] 国际目的地已包含签证和保险信息
- [ ] 欧洲目的地已包含联票策略和交通方案
- [ ] 北美目的地已包含自驾信息和 Pass 策略
